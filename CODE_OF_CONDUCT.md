# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be respectful. Disagree with ideas, not people.
- No harassment, discrimination, or personal attacks.
- Assume good faith; ask before assuming the worst.
- Maintainers may remove comments, commits, code, issues, and contributions that aren't aligned with this Code of Conduct.

Report unacceptable behavior to **conduct@cyberunite.com**. All complaints will be reviewed and investigated promptly and fairly.
